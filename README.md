
# Main Project: Investment gold strategy

In the face of geopolitical tensions and market uncertainty, gold is often considered a safe-haven asset and protection against crisis. This project studies how moving average can be used to profit of the current economy and geopolitical context. 

**Why gold ?**

-High volatility

-High volume implying liquidity 

-Strong momentum perfect for breakouts

-Accessible assets compared to real institutional tools








**Objectives:**

-Collect and **prepare gold market time series**.

-**Build and compare a simple MA system**  and a ML-assisted version to filter false signals.

-**Implement backtests and performance metrics** to evaluate robustness.

-**Discover new AI** specially adapted to the finance environment.





_Educational Note: This project is intended for academic purposes and does not constitute financial advice. Simulations are conducted for study purposes only._
